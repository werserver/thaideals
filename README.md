A lightweight, high-performance affiliate shop template designed for easy cloning across multiple domains.

## Features

- **Multi-Domain Ready**: Easily export/import configurations via the Admin Panel.
- **Data Sources**: Supports both API (Passio/Ecomobi) and CSV files.
- **URL Cloaking**: Built-in support for URL cloaking via `goeco.mobi`.
- **Customizable Identity**: Change site name and favicon directly from the Admin Panel.
- **SEO Optimized**: Dynamic meta tags, JSON-LD, and sitemap support.
- **Modern UI**: Built with React, Tailwind CSS, and Shadcn UI.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start development server: `npm run dev`.
4. Access Admin Panel at `/admin` (Default: admin / sofaraway).

## Deployment

This project is optimized for deployment on **Vercel**. Simply connect your GitHub repository and it will handle the rest.
